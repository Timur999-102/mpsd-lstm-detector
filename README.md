# 🔍 PowerShell Malware Detector (LSTM)

Нейросеть для анализа обфусцированного PowerShell-кода на вредоносность.

## 🛠️ Технологии
- TensorFlow/Keras — LSTM архитектура
- ClearML — логирование экспериментов и управление данными
- scikit-learn — предобработка и метрики

## 🚀 Быстрый старт
```bash
pip install -r requirements.txt
python src/train_lstm.py
